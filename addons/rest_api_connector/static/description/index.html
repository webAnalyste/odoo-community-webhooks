REST API Connector

API REST JSON pour piloter Odoo depuis n8n, Make, scripts, etc.

Authentification

Toutes les requêtes nécessitent le header X-API-Key: votre_cle

Configurer la clé : Paramètres → Technique → Paramètres système → clé rest_api_connector.api_key

Filtres — syntaxe


Double underscore obligatoire : champ__like, champ__gte, champ__lte



  

    SyntaxeOpérateurExemple

champ__like=valeurContient (ilike)name__like=veolia / partner_name__like=veolia
