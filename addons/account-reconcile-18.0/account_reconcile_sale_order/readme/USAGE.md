To use this module, you need to:

1. Have a payment on a bank statement matching the amount of an *invoiceable* sales order
2. Enter the reconciliation screen
3. Observe that the sales order is offered as reconciliation counterpart
4. You can manually select and deselect sales orders in the *sales orders* tab of the reconciliation widget

Note the reconciliation only works if fully invoicing the sales order yields an invoice over the order's total amount. Usually this means that all products in the sales order must have invoicing policy *Ordered quantities*.
